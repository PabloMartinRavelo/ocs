QPT 1.0.28 Release Notes

QPT 1.0.28 is a bug fix release for the previous version (which was actually v1.0.26). Release notes for the previous release are available here.

The QPT documentation index is available here.

To install and run this release of QPT:
First, uninstall any existing 2012A version → launcher -a:qpt-2012A -u
To install → launcher -i:qpt -a:qpt-2012A -h:build.cl.gemini.edu
To run → launcher -a:qpt-2012A
To uninstall → launcher -a:qpt-2012A -u
As always, the first time you run QPT it will download necessary components. This may take several minutes, especially in the north. Your previous installation of QPT should continue to function normally.

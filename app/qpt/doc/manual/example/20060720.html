NOTES ABOUT THE QPT
Please ignore warnings saying "Variant conditions better than necessary". This means the program could be executed in worse conditions than those for which it is scheduled.
In the elevation plots for each plan, if an observation is green that means it is a perfect match for the conditions. Yellow or orange means there is some problem and the warning message will tell you what the (potential) problem is.
SPECIFIC INSTRUMENTS NOTES:

GNIRS: calibrations by the end of the night, EXCEPT:
  • IFU that must be run before the grating is moved (after the science observation)
  • R=18000 mode (LongCamera, 111l/m, 0.1arcsec slit) that must be done before slewing the telescope after the science observation
  • in case the PI actually asked for it
  • Also:
    • run the calibrations from within the programme
    • at the end of the night, do first the non-saturating calibrations (long slit), then the XD ones.
    • for the saturating flats, go from smaller to larger slits.
GMOS-S:
  • Observer u,g,[OIII] imaging twilight flat at the evenining
  • r',i',z'.Ha,SII twilight flats (images) must be observed at morning
  • Don' forget to observe flux standards when during bad weather
  • Check the GMOS-S calibrations file for needed GMOS-S night time calibrations
  • Plese read the GMOS-S Nighttime Observing Procedures to know how to observe twilight flats (imaging and spectroscopy).

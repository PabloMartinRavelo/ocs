QPT 1.0.5 Tabano Release Notes

QPT 1.0.5 Tabano is simply an update to 1.0.5 for compatability with the Tabano release of OCS.
Release notes for the previous release are available here.

The QPT documentation index is available here.

To install this release of QPT, please install and run it using an alias as follows.
To install → launcher -h:sbfmaps.cl.gemini.edu -i:qpt -a:qpt-1.0.5-tabano -v
To run → launcher -a:qpt-1.0.5-tabano
To uninstall → launcher -a:qpt-1.0.5-tabano -u
As always, the first time you run QPT it will download necessary components. This may take several minutes, especially in the north. Your previous installation of QPT should continue to function normally.

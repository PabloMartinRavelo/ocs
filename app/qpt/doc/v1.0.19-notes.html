QPT 1.0.19 Release Notes

QPT 1.0.19 addresses the issues described below. Also, please note the new download site indicated in the installation instructions. Release notes for the previous release are available here.

The QPT documentation index is available here.

To install and run this release of QPT:
First, uninstall the existing 2011B version → launcher -a:qpt-2011B -u
To install → launcher -i:qpt -a:qpt-2011B -h:build.cl.gemini.edu
To run → launcher -a:qpt-2011B
To uninstall → launcher -a:qpt-2011B -u
As always, the first time you run QPT it will download necessary components. This may take several minutes, especially in the north. Your previous installation of QPT should continue to function normally.

New and Notable

SCI-0448: Daily calibration and engineering observations display is not affected by band filters.

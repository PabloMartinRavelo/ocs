Browsing the Database using the Observing Log Web Application

The Observing Log application provides three different views of completed observations

  • The Plan View is the view of observations taken on a single night in the order they are acquired. In Plan View there is one line for each unique configuration of the instrument used to acquire data.
  • Observation View is the view of a single observation showing all the observation's datasets. The configuration of the system for each dataset is completely shown.
  • Program View shows all the observations in a Science Program. As with Plan View, the observations are shown as one line for each unique configuration.

Using the Action links

  • Selecting the obs link in Program View or Plan View shows the Observation View for a row.
  • Selecting the prog link in Plan View or Observation View shows the Science Program View that a row.
  • Selecting the plan link in Program View or Observation View shows the Plan View for the specific row.
